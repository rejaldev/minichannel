# MiniChannel Frontend

Web Dashboard untuk MiniChannel POS System.

## Tech Stack

- Next.js 16.1.1 (App Router + Turbopack)
- React 19.2.1
- TypeScript 5
- Tailwind CSS 4
- Socket.io Client

## Quick Start

```bash
npm install
npm run dev
```

Server: http://localhost:3100

## Pages

- `/login` - Authentication
- `/pos` - Point of Sale
- `/dashboard` - Overview
- `/dashboard/products` - Product management
- `/dashboard/stock` - Stock & alerts
- `/dashboard/transactions` - Transaction history
- `/dashboard/settings` - System settings

## Build

```bash
npm run build
```

Deploy ke Vercel atau static hosting.
